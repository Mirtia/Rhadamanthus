uint8_t idt_entry64::type_attr

bits 3–0: gate type (0xE = 64-bit interrupt gate, 0xF = 64-bit trap gate).

Gate type/attributes: bit 7: P (Present); bits 6–5: DPL; bit 4: 0 (must be zero for interrupt/trap gates);
