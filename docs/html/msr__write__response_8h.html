◆ MSR_IA32_LSTAR

#define MSR_IA32_LSTAR    0xC0000082

Common security-relevant MSR indices.

───────────────────────────────────────────── JSON Response Structure (msr_write_response) ───────────────────────────────────────────── { "event": "MSR_WRITE", "vcpu_id": 0, "regs": { "rip": "0xfffff80000010234", "rsp": "0xfffff8000010ff00", "cr3": "0x0000000123456000" }, "msr": { "index": "0x00000000c0000082", "value": "0xfffff80000234567", "name": "IA32_LSTAR", } }

Security-relevant MSR indices

These MSRs are commonly targeted by rootkits and kernel-level malware for system call interception, privilege escalation, and security bypass. All are monitored by modern OS security mechanisms like PatchGuard. See: https://www.intel.com/content/www/us/en/content-details/782158/intel-64-and-ia-32-architectures-software-developer-s-manual-combined-volumes-1-2a-2b-2c-2d-3a-3b-3c-3d-and-4.html 64-bit SYSCALL entry point (most critical) \ Contains RIP for SYSCALL in long mode \ Rootkits modify this to intercept ALL system calls \ Protected by PatchGuard, monitored every ~30 minutes
