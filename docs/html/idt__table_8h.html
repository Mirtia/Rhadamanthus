void *  context 
)

Callback that inspects IDT entries and flags handlers outside kernel text.

The function resolves _stext and _etext to bound the expected kernel text region, reads the IDTR base of vCPU 0, loads vector names from disk, walks all 256 vectors, and logs any named handlers whose addresses fall outside [_stext, _etext].

Parameters
vmiThe LibVMI instance.
contextUser-defined context [unused].
Returns
VMI_SUCCESS on successful inspection else VMI_FAILURE.
