Detailed Description

Event payload for a kallsyms table write.

───────────────────────────────────────────── JSON Response Structure (kallsyms_table_write_response) ───────────────────────────────────────────── { "event": "KALLSYMS_TABLE_WRITE", "vcpu_id": 0, "regs": { "rip": "0xfffff80000010234", "rsp": "0xfffff8000010ff00", "cr3": "0x0000000123456000" }, "memory": { "write_gla": "0xfffff80000234567", "write_gpa": "0x0000000012345678" } }

The kallsyms table contains kernel symbol information used for debugging, stack traces, and symbol resolution. Modifications to this table are extremely suspicious and typically indicate rootkit activity attempting to hide kernel symbols or manipulate symbol resolution mechanisms.
